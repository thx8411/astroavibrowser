Menus

File Menu

Open...

Opens a video file in any ffmpeg/libav supported format.

Save...

Saves the frames you selected using either the BMP or the AVI output, depending on the output and the codec you choosed.

Save <plan>...<>

Saves the frames you selected like the usual 'save' menu, but only stores one of the color plans.

  • Luminance : frames are converted to grey level pictures
  • R plan : red frames components converted to grey level pictures
  • G plan : green frames components converted to grey level pictures
  • B plan : blue frames components converted to grey level pictures

Build <method> <colors>

Saves the selected frames applying the selected method, either like a grey level picture or an RGB picture. Pictures are saved as BMP files.

method
  • sum : summing pixels of each frames
  • max : just keep the bightest pixel values of all frames
  • mean : resulting pixel is the mean of all the selected frames pixels
  • median : resulting pixel is the mean of all the selected frames pixels
colors
  • (mono) : luminance only, 8bits picture
  • (rgb) : RGB color picture

Menu Settings

Force RGB24 output

Some software doesn't support the AVI raw 8 bits format (Y800). Using this option, you may force every raw output to RGB24.

Notes

  • In raw format, 'avi' files should be stored upside-down, using a negative height. 'gstreamer' 0.10 doesn't follows this specification. In order to be compatible with 'gstreamer', astrovaibrowser stores raw movies upside-up, with a positive height. Depending on the software you use, your output video may be verticaly fliped.
